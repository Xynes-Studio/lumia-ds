# Calendar

The Calendar component is a flexible date picker that supports single date and date range selection. It comes with two variants: `compact` (default) for dropdowns/popovers and `full` for full-screen management interfaces.

## Features
- **Variants**: Compact and Full-screen modes.
- **Selection Modes**: Single date, Date range.
- **Customization**: Fully customizable day rendering via `renderDay`.
- **Navigation**: Interactive header with month/year selection.
- **Localization**: Built on top of `date-fns` and supports localization.
